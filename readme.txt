woo commerce's Addon plugin. show notification to visitors about recent purchased items.

== Description ==

Product-Purchase-Notifications works with woo commerce.on activation it start showing notifications about recent purchases across the globe.
The notification contains following informations.
-Product name
-Buyer name
-Product thumbnail
-City where product is purchased


= Customizable =
 
This plugin support full customization 
-Notification delay time
-Notification location (left/right)
-Notification background color
-Notification text color 
-Product thumbnail (show/hide)
-Buyer Name (show/hide)


== Installation ==

1. Upload the entire `Product-Purchase-Notifications` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find a **setting tab** under Settings menu page you can customize all of the customization here.you notification will be start showing on front end.
